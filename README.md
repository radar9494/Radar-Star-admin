Radar-Start 开源数字货币交易系统
=====================================

    Radar-Start Java区块链平台，基于Springboot开发的区块链平台。

## 项目介绍

    帮助用户简单、快捷、低成本地进行支付、转账和全球货币的自由兑换

### 平台使用的技术
- 后端：Spring、SpringMVC、Mybatis、SpringBoot

- 数据库：Mysql

- 其他：redis、activeMq、fastdfs、谷歌验证码等等

### 系统运行环境
- Centos7.4.5
- MySQL5.7
- Redis
- acativemq
- nginx
- JRE 8u261
- JDK 1.8
